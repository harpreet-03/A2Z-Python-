
Python can be used to perform operations on a file. (read & write data)

# Open, read, & close files

-> We've to open a file before reading or writing

f = open("file_name", "mode")

Mode:
r: read mode
w: write mode

-- after then, read and close the file ::::::::::
data = f.read()
f.close()
